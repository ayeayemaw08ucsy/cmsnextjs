## cmsnextjs
Content Management System with Nextjs Reactjs Material ui Mongo

Run Command 
  DEVELOPMENT
     -- npm install
     -- npm run client
     -- npm run server
  Docker Deployment
     -- docker-compose up --build
  Setup .env
       change the value in env file according to docker file.
       -SVR_PORT
       -CLI_PORT
       -ROOT_URL
       -MONGODB_URI
